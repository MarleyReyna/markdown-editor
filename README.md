# Hey!

## This app allows you to:
1. Create markdown files and preview them
2. Save changes and delete files
3. Toggle between light and dark mode
4. Rename files
5. View markdown previews in fullscreen mode

## This app was built with:
- React
- Javascript
- SCSS
